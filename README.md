# Deadline Bot

Telegram-бот для дедлайнов учебной группы. Бот принимает дедлайны от пользователей из whitelist, публикует их в канал, отправляет напоминания, поддерживает отмену, удаление и архив.

## Быстрый старт

Нужен Python 3.11+.

1. Установите зависимости:

```bash
pip install -r requirements.txt
```

2. Заполните `.env`:

```env
TOKEN=ваш_токен_бота
CHANNEL_ID=@username_канала_или_-100xxxxxxxxxx
WHITELIST_USER_IDS=123456789,987654321
DEADLINES_STORAGE_PATH=/абсолютный/путь/к/deadlines.json
```

3. Запустите бота:

```bash
py app.py
```

По умолчанию данные лежат в `deadlines.json` рядом с кодом. Для сервера лучше использовать `DEADLINES_STORAGE_PATH`, чтобы хранить состояние вне репозитория.

## Что умеет бот

- Принимает дедлайны только от пользователей из `WHITELIST_USER_IDS`
- Формат даты: `DD.MM.YYYY` или `DD.MM.YYYY HH:MM`
- Использует фиксированный часовой пояс `UTC+5`
- Публикует дедлайн сразу, если до него 7 дней или меньше
- Иначе спрашивает, делать ли первую публикацию сразу
- Напоминает за 7 дней и за 24 часа
- Позволяет изменить, отменить и удалить дедлайн
- Хранит архив неактивных дедлайнов
- Через 3 дня после завершения или отмены удаляет все связанные сообщения из канала

## Интерфейс

Основной интерфейс кнопочный:

- `Добавить дедлайн`
- `Список дедлайнов`
- `Архив`
- `Изменить дедлайн`
- `Отменить дедлайн`
- `Удалить дедлайн`
- `Пропустить`

Дополнительно есть команды:

- `/now` — показать текущее время глазами бота
- `/archive` — показать архив

## Формат дедлайна

С временем:

```text
Описание
13.04.2026 18:30
```

Без времени:

```text
Описание
13.04.2026
```

Если время не указано, логически используется `00:00`, но в тексте дедлайна оно не показывается. Если пользователь явно ввёл `00:00`, бот его показывает.
